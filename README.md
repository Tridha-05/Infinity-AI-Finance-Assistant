# AI Finance Assistant

A Streamlit-based data analytics application that analyzes personal financial transactions
and provides visual insights and NLP-based summaries.

## What this project demonstrates
- Data preprocessing and analysis using Pandas
- Exploratory data analysis and visualization using Plotly
- Use of a pretrained NLP model for sentiment-based financial summaries
- Building interactive data apps using Streamlit

## Features
- Upload CSV transaction data or use simulated transactions
- Category-wise expense analysis
- Interactive charts for spending trends
- Sentiment-based financial health summary (pretrained model)
- Demonstration-only encryption for storing summaries

## Tech Stack
- Python
- Pandas, NumPy
- Plotly
- Streamlit
- HuggingFace Transformers (pretrained model)

## Disclaimer
This project is for educational purposes only.
It uses mock data and demonstration-level encryption.
